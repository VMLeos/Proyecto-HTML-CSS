Bordes en HTML con CSS

En este apartado veremos cómo crear bordes en una página HTML con CSS

Este texto está contenido entre un borde dotted. Para ello usaremos:
p {
border: #0973D2 10px dotted;
}


Este texto está contenido entre un borde dashed. Esta vez usaremos:
p{
border: red 5px dashed;
}

Este texto está contenido entre un borde double.
Vamos a usar:
p
{
border: rgb(23,206,38) 20px double;
}

Bordes con relleno

Este texto está contenido entre un bordegroove. Para esto usaremos:
p {
background-color: #000;
border: white 12px groove;
color: #fff
}
