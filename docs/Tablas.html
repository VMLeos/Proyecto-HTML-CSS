Elaboración de una tabla simple

Para la tabla simple usaremos el comando < tr > y para nombrar cada elemento usaremos el comando < td > < table style="width: 100%" >

Elaboración de una tabla con bordes

Para la tabla con bordes usaremos el comando < tr > y para nombrar cada elemento nuevamente usaremos el comando < td > < table style ="width: 100%" border ="" >

Títulos de tabla

Para centrar y distinguir los títulos en la tabla usaremos los comandos < thead > en el encabezado de la tabla, y cada elemento entre < th > centrando cada elemento del encabezado, mientras que "border= X" nos va a dar el ancho del borde.

Combinar columnas en la tabla

Para combinar dos o más columnas bajo un encabezado, usaremos el comando < th colspan="X" > en donde X es el número de columnas a unir

Combinar filas en la tabla

Para combinar dos o más filas bajo un mismo encabezado, usaremos el comando < th rowspan="X" colspan="X" > en donde X es el número de columnas y filas a unir
