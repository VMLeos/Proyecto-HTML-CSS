Listas

Esta página muestra cómo hacer listas en HTML



Listas desordenadas

En estas listas no importa el orden de los elementos y cada uno se muestra como viñeta
Se usa el comando < ul > para este tipo de lista, y cada elemento va entre el comando < li >

  • Ferrari
  • Audi
  • Volk's Wagen
  • Ford
  • Chevrolet
  • Honda

Listas ordenadas

En estas listas si importa el orden de los elementos y se muestran numeradas, ya que llevan cierto orden
Se usa el comando < ol > para este listado, y de la misma manera cada elemento va entre el comando < li >

  1. Mercurio
  2. Venus
  3. Tierra
  4. Marte
  5. Jupiter
  6. Saturno
  7. Urano
  8. Neptuno
  9. Pluton

Listas descriptivas

Estas listas nos ayudan a hacer definiciones o descripciones.
Se forma con el comando < dl> y los elementos se enlistan entre < dt>
mientras que los atributos del elemento se enlistan con < dd>.

Coca-Cola
Sabor cola
Color café oscuro
Botella verde
Fanta
Sabor naranja
Color anaranjado
